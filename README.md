# Darsi

Darsi is an online Fullstack ShadcnUI + React + Shadcn ui learning and course platform

## Requirements
- Bun
- Typescript
- ShadcnUI
- React

### Installing

1. Open a powershell terminal in administrator mode
and execute the following command

```pwsh
powershell -c "irm bun.sh/install.ps1|iex"
```

2. Once bun is installed restart your terminal and double check bun is installed and present in the environment by executing the following command


```pwsh
bun
```

if the command fails restart your computer and check again

3. Install all the dependencies with

```pwsh
bun install
```

4. Now you can run the project by running the command

```pwsh
bun run dev
```

### Contributing

In order to contribute you need git and access to a github account
1. install git and fork the repository
2. Clone the forked repository and apply changes
3. Commit changes and push them to your branch
4. Submit a pull request to the initial repository
